## Цветовая схема

### Статусы участников
- 🤔 Думает - серый (#6B7280)
- ✅ Проголосовал - зеленый (#10B981)

### Календарь
- Обычный день - белый фон, серая рамка
- Выбранный день - синий фон (#2563EB), белый текст
- Сегодня - голубая рамка (#60A5FA)
- Недоступный день (прошедший) - серый (#9CA3AF)

### Интерактивные элементы
- Кнопки - синий (#2563EB)
- Ссылки - синий (#2563EB)
- Hover состояния - темнее на 20%

## Адаптивность

### Breakpoints
- Mobile: < 768px
- Tablet: 768px - 1024px
- Desktop: > 1024px

### Mobile особенности
- Боковая панель участников сворачивается в выдвигающееся меню
- Календарь занимает всю ширину
- Увеличенные размеры кнопок для touch интерфейса
- Вертикальная ориентация элементов

### Tablet особенности
- Уменьшенная боковая панель
- Более компактный календарь
- Двухколоночная layout для некоторых элементов
